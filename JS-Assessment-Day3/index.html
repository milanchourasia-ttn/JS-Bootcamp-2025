Q3.Explain difference between Bind and Call (example).

Bind method

The bind method returns a new function with the specified this context, without invoking function immediately.

        const person = {
        name: "Shivang",
        personFunction: function (arg) {
            console.log(`${arg} I am ${this.name}`);
        },
        };
        const person2 = {
        name: "Shreeja",
        };
        const greetBind = person.personFunction.bind(person, "Hello");
        greetBind();

Call method

The call method invokes the function immediately with a specified this context and arguments passed individually.

        const person = {
        name: "Shivang",
        personFunction: function (arg) {
            console.log(`${arg} I am ${this.name}`);
        },
        };
        const person2 = {
        name: "Shreeja",
        };
        person.personFunction.call(person2, "Hi");
